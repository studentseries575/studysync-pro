# 🚀 StudySync Pro Dashboard

A modern productivity web application designed for students to manage tasks, notes, and reading progress efficiently.

---

## 🌟 Features

* ✅ Task Management (Add, Edit, Delete, Filter)
* 📝 Notes System
* 📖 Book Tracker
* 📅 Calendar with task highlights
* 🔔 Notifications (Due & Overdue)
* 📊 Charts & Analytics
* 🌙 Dark Mode
* 👤 User Profile Customization

---

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript
* Firebase Authentication
* Firestore Database
* Chart.js
